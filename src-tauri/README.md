# TK Dev Tools

Migration scaffold for the Tauri version of the TK Dev Tools suite.

## Current focus

- React shell for the app architecture map
- Rust command contract for the main overview screen
- A foundation that will absorb the Python GIF generator next

## Planned modules

- Core parsing and rendering in Rust
- Tauri commands for file selection, generation, logs, and integrity checks
- React workspace for the generator UI and future tools

## Run

```bash
npm install
npm run tauri dev
```
